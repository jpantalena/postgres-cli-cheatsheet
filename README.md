# postgres-cli-cheatsheet
Cheatsheet of PostgreSQL CLI commands - psql.

## Start psql
```
psql
```

## List of databases
```
\l
```

## Create a database
```
create database <database-name>
```

## Delete a database
```
drop database <database-name>
```

## Connect to a database
```
\c <database-name>
```
